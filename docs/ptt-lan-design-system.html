Design System — PTT-LAN

Paleta sóbria e superfícies escuras de baixo contraste, pensadas para uso rápido de walkie-talkie: o olho deve achar o botão de PTT e o status de quem está falando sem esforço. Cor é usada como sinal de estado (transmitindo, recebendo, conectado), nunca como decoração.

Aplicação em código (core-designsystem): estes tokens devem ser expressos como Color, Typography e Shape do tema Material 3 em Compose Multiplatform, nunca como valores de cor hard-coded dentro de módulos feature-*. Estados de cor (idle / transmitindo / recebendo / online / offline) devem ser modelados como um enum de domínio mapeado para cor apenas na camada de apresentação.
